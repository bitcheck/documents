# 二、应用场景1：红包
### 1- 为什么要做链上红包？
区块链公链发展至今，行业内一直在做各种“出圈”的努力，希望有越来越多的圈外朋友使用钱包以及DAPP应用，红包是一种比较容易让新生事物迅速普及的手段，所以我们开发了红包功能。

### 2- 链上红包与微信红包有什么区别？
微信红包是社交属性较强的红包产品，`bitCheck`区块链红包的营销属性较强（如通过留言打广告）。微信群红包经常会遇到抢群红包机器人，导致红包的作用大打折扣，而链上红包的参与人比较精准，用户抢红包时需要学会钱包以及DAPP的基本操作，使得在分享红包的同时培育了用户。

### 3- 为什么要在火币生态链上首发红包？
红包作为一种小额支付应用，如果GAS手续费高于收取小额红包时的所得，红包将失去意义。同时，还需要抢红包时快速的体验。目前，火币生态链满足这个需求，所以在该链上首发红包功能。

### 4- 链上红包使用过程中需要注意什么？
<font color="red">因为区块链红包不能确定收款方，强烈建议红包金额不易过大。为安全起见，合约已经对每个红包金额做了相当于50~100USDT的限定。</font>

### 5- 什么是红包衰减系数（Cliff）？
在发红包时，除了金额和留言外，需要设置一个叫“衰减系数”的参数。

假设某用户发了10USDT的红包，衰减系数为10%，则第一个抢到的可得10*10%=1USDT；第二个抢到的得到剩余9USDT中的10%，即0.9USDT；第三个抢到的得到剩余8.1USDT中的10%，即0.81USDT，以此类推，先抢到的得到的多。只要发红包的用户不将此红包撤回，可以无限量抢红包。

如果将衰减系数设为100%，则只发一个红包。

### 9- 什么是发红包挖矿？
发红包挖矿就是在`易快信支付（BitCheck）`应用中，发出红包，每次被收取时，发红包的用户会收到平台币`BTCH`的奖励。

### 10- 如何计算挖到的BTCH数量？
计算公式如下：
```
挖得BTCH = 收取红包金额^(2/3) * 基础系数 * 时间差系数 * 阶段系数 * (1-矿税)
```
* 基础系数 = 0.05
* 时间差系数：参考《BitCheck经济模型绿皮书》时间差系数部分，如下表：

|时间间隔|系数|
   |:-|-:|
   |<1小时|0|
   |1-24小时|0.05|
   |24-48小时|0.15|
   |48-96小时（2-4天）|0.168|
   |96-192小时（4-8天）|0.206|
   |192-384小时（8-16天）|0.286|
   |384小时-720小时（16-30天）|0.455|
   |>720小时（30天以上）|0.815|

* 阶段系数 = 10
* 矿税 = 20%

所以上面公式可以简化为：
```
挖得BTCH = 收取红包金额^(2/3) * 0.5 * 时间差系数 * (1-20%)
```

注：
* 以上公式与系数详解，请参考[《BitCheck经济模型绿皮书》](http://defimao.cn/?p=955 "《BitCheck经济模型绿皮书》")
* 与常规支付挖矿不同，发红包挖矿模式下，只有发红包的用户可以获得BTCH，收红包的用户无法得到BTCH。

#### 11- 红包费用
收取红包时，会被扣除`0.5%`作为费用。

#### 12- 未领取的红包余额如何处理？
发送红包的账户选择：发现-我发的红包-选择灰色垃圾桶按钮，即可回收尚未领取的红包。

# 三、应用场景2：挖矿

`BTCH`的介绍与奖励算法，参考[《BitCheck经济模型绿皮书》](http://defimao.cn/?p=955 "《BitCheck经济模型绿皮书》")

挖矿基于支付行为，支付用法参考《应用场景4：电子商务担保支付》，本篇仅介绍与挖矿相关的操作。

### 1- 什么时候能获得BTCH？
用户在完成一个完整的支付交易（收款方收款），且付款与收款的时间间隔超过1小时的，收付双方同时获得`BTCH`。

### 2- 要获得BTCH，有没有最低支付金额的要求？
有，支付金额在10USDT以上，才能获得`BTCH`。

低于10USDT，您仍旧可以完成正常的支付，但是没有`BTCH`。

### 3- 收付双方获得的奖励如何分配？
有两种分配方式：
* 付款方不是超级节点（即不拥有`BCT`），双方各分得`50%`。
* 付款方为超级节点，双方分配比例由超级节点设定。

### 4- 什么是超级节点？
超级节点是参与`bitCheck`日常治理的去中心化组织成员，每个超级节点拥有一枚`BCT`代币，总量`21`枚，将采取链上拍卖的方式分发。

### 5- 超级节点有什么权利？
* 参与社区治理投票。
* 在公测期内，每枚`BCT`拥有`5倍`于常规挖矿的收益。挖出的`BTCH`将在180天内每秒线性释放，公测期总量为150万个`BTCH`。
* 超级节点可以在DAPP中自行设定付款方与收款方获得挖矿奖励的分配比例。

### 6- 如何提取BTCH奖励？
有两种方式：
* 付款方不是超级节点（即不拥有`BCT`），收款方在提现后，`BTCH`自动划转到双方账户。
* 付款方为超级节点，双方获得`BTCH`有180天的解锁期，所以需要双方自行提取。提取时间间隔可随意，每秒钟都会释放。

### 7- 如何提取解锁的BTCH？
在DAPP中，点击【发现】，选择【超级节点管理】，点击【提取】按钮，如下图：
<img src="https://tva1.sinaimg.cn/large/008eGmZEgy1gn5kedv0xaj30ki0o6q4p.jpg" width=360/>

### 8- 如何获得5倍的超额挖矿收益？
加入超级节点社群，让超级节点作为付款方与你交易。

### 9- 如何参与手续费分红？
在`bitCheck`进行支付收取的交易手续费中的`2/3`用于分红，每周末，所有`BTCH`持有者可根据持有的份额，提取分红。

每周分红时间为周五中午12点至周日中午12点。

在每周开始分红前，需要将`BTCH`打入分红池，分红池会根据每个账号在分红池内`BTCH`的占比，将手续费返还给这些`BTCH`持有人。

在DAPP中，点击`发现`，选择`BTCH分红`，输入需要存入的`BTCH`数量，然后点击`存入BTCH`按钮。

注：
* 分红金额会在分红提取开始一瞬间根据当时分红池的实际数量显示。
* 分红仅限于第一阶段，第一阶段720万枚`BTCH`产生后，将停止分红，采用回购销毁机制。

### 10- 如何提取分红？
每周的分红时间到后，在DAPP中，点击`发现`，选择`BTCH分红`，点击`提取分红`按钮。

### 11- 超级节点如何设定首付款双方的分配比例？
在DAPP中，点击`发现`，选择`从矿池提取BTCH`。

如果你是超级节点，即`BCT`持有人，可看到以下调整分配比例的操作：
